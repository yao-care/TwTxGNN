## 競品詳細分析

### 競品 A：TxGNN (Harvard Zitnik Lab)

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://zitniklab.hms.harvard.edu/projects/TxGNN/ |
| 定位 | AI 藥物再利用基礎模型研究專案 |
| 目標受眾 | 學術研究人員、AI/ML 研究者 |
| 語言 | 英文 |

#### 內容策略

| 項目 | 觀察 |
|------|------|
| 資訊架構 | 單頁展示：模型介紹 → 論文 → 程式碼 → 工具 |
| 主要內容類型 | 學術論文、GitHub 程式碼、研究展示 |
| 更新頻率 | 低（學術發表週期） |
| 特色內容 | Nature Medicine 論文、Human-AI Explorer 工具 |

#### 優勢與特色

1. **學術權威性**：哈佛醫學院背書，Nature Medicine 發表
2. **完整程式碼**：GitHub 開源，可重現
3. **配套工具**：txgnn.org 提供互動式探索界面

#### 可改善之處

1. **無本地化**：僅英文，無亞洲語言版本
2. **無證據整合**：只有預測分數，無臨床試驗/文獻彙整
3. **靜態展示**：網頁缺乏互動性

## 差異化建議

基於競品分析，建議我們的差異化方向：

### 核心差異化（必須強調）

1. **「預測 + 證據」完整報告**
   - 競品只有預測分數，我們有證據整理
   - Slogan：「不只告訴你可能有效，還告訴你證據在哪裡」

2. **台灣健保藥品聚焦**
   - 唯一整合 TFDA 資料的平台
   - 繁體中文內容，服務台灣研究社群

3. **可行動的決策建議**
   - L1-L5 證據等級 + Go/Proceed/Consider 建議
   - 降低研究人員的決策成本

### 次要差異化（可強化）

4. **DDI 安全性整合**
   - 222,391 筆藥物交互作用資料
   - 安全性是其他平台未涵蓋的面向

5. **YMYL 合規**
   - 明確免責聲明、資料來源透明
   - 建立醫療健康內容的專業形象
