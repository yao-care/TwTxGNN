## 差異化建議

基於競品分析，建議我們的差異化方向：

### 核心差異化（必須強調）

1. **「預測 + 證據」完整報告**
   - 競品只有預測分數，我們有證據整理
   - Slogan：「不只告訴你可能有效，還告訴你證據在哪裡」

2. **台灣健保藥品聚焦**
   - 唯一整合 TFDA 資料的平台
   - 繁體中文內容，服務台灣研究社群

3. **可行動的決策建議**
   - L1-L5 證據等級 + Go/Proceed/Consider 建議
   - 降低研究人員的決策成本

### 次要差異化（可強化）

4. **DDI 安全性整合**
   - 222,391 筆藥物交互作用資料
   - 安全性是其他平台未涵蓋的面向

5. **YMYL 合規**
   - 明確免責聲明、資料來源透明
   - 建立醫療健康內容的專業形象

#### 資料規模

| 項目 | 數量 |
|------|------|
| 藥物 | 49,652 |
| 治療靶點 | 4,221 |
| 癌症特徵簽名 | 25 |
| 文獻來源 | 134 篇（2000-2023） |
| 已驗證再利用藥物 | 169 |

#### 18 種預測演算法詳解

DrugRepoBank 的預測模組採用 **4 大類、18 種演算法**：

##### 1. 相似性方法 (Similarity-based)

| 方法 | 原理 |
|------|------|
| 藥物-藥物相似性 | 結構相似的藥物可能有相似療效 |
| 靶點-靶點相似性 | 功能相似的靶點可能被相同藥物調控 |

**原理**：如果藥物 A 和藥物 B 化學結構相似，且 A 能治療疾病 X，則 B 可能也能治療 X。

##### 2. AI 方法 (Artificial Intelligence-based)

| 方法 | 原理 |
|------|------|
| 機器學習 | 從已知藥物-靶點關係學習模式 |
| 深度學習 | 神經網路預測藥物-靶點交互作用 |

**原理**：訓練模型學習「哪些藥物特徵會與哪些靶點結合」，然後預測新的配對。

##### 3. 簽名方法 (Signature-based)

| 方法 | 原理 |
|------|------|
| 基因表達簽名反轉 | 候選藥物應「反轉」疾病的基因表達變化 |

**原理**：
- 疾病使某些基因上調（↑）、某些下調（↓）
- 好的候選藥物應該讓上調的基因下調、下調的基因上調
- 這就是「Connectivity Map」的核心概念

##### 4. 網路方法 (Network-based)

| 方法 | 原理 |
|------|------|
| DRviaSPCN 等 | 基於生物網路中的子通路交叉評估藥物-疾病關聯 |

**原理**：建立藥物-靶點-疾病的知識圖譜，分析網路中的路徑關係。這也是 **TxGNN 的核心方法**。

##### DrugRepoBank vs TwTxGNN 方法比較

| 項目 | DrugRepoBank | TwTxGNN |
|------|--------------|---------|
| 演算法數量 | 18 種（ensemble） | 1 種 (TxGNN) |
| 藥物數量 | 49,652 | 191（台灣健保聚焦） |
| **證據整合** | ❌ 僅預測分數 | ✅ 臨床試驗 + 文獻 |
| **證據等級** | ❌ | ✅ L1-L5 |
| **決策建議** | ❌ | ✅ Go/Proceed/Consider |

**結論**：DrugRepoBank 的優勢是演算法多樣性（18 種方法），適合探索各種預測方法。TwTxGNN 的優勢是證據深度：不只給預測分數，還整合臨床試驗、文獻、DDI，並給出可行動的決策建議。

#### 優勢與特色

1. **多演算法整合**：18 種方法，涵蓋相似性、AI、簽名、網路
2. **雙模組設計**：文獻（實驗證據）+ 預測（演算法）
3. **互動式介面**：藥物-靶點-疾病-副作用網路視覺化
4. **生物資訊分析**：疾病簽名分析功能

#### 可改善之處

1. **無證據等級**：預測結果無 L1-L5 分級
2. **無決策建議**：無 Go/Proceed/Consider 建議
3. **無本地化**：無台灣藥品資料、無中文
4. **介面複雜**：學習曲線較高

## 競品 I：ReFRAME / reframeDB (Scripps Research)

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://reframedb.org/ |
| 定位 | 開放式藥物再利用篩選庫 |
| 背景 | Scripps Research / Calibr，Bill & Melinda Gates Foundation 支持 |
| 規模 | 13,288 化合物 |

#### 資料來源

ReFRAME 庫整合三大商業藥物資料庫：
- Clarivate Integrity
- GVK Excelra GoStar
- Citeline Pharmaprojects

並透過專利探勘補充已在人體使用的小分子。Calibr 團隊購買約 7,000 化合物，並由約 500 位化學家在 18 個月內合成約 5,000 化合物。

#### 優勢與特色

1. **開放存取政策**：遵循 Gates Foundation 全球存取政策，所有篩選結果公開

## 競品 K：CLUE / Connectivity Map (Broad Institute)

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://clue.io/cmap |
| 定位 | 基因表達簽名匹配平台 |
| 背景 | Broad Institute，NIH LINCS 計畫 |
| 規模 | 7,000+ 基因表達譜，涵蓋 1,309 化合物 |

#### 核心方法

**Connectivity Map 原理**：
1. 使用者提供「基因命中清單」（疾病相關的差異表達基因）
2. 系統比對大規模的藥物誘導基因表達資料庫
3. 排名出「能逆轉疾病基因表達變化」的候選藥物

這是一種 **Signature-based** 方法，與 DrugRepoBank 的簽名方法類似。

#### 優勢與特色

1. **雲端運算平台 CLUE**：免安裝，網頁操作
2. **整合 LINCS 資料**：超過 150 萬組基因表達實驗
3. **COVID-19 應用**：已用於識別抗 SARS-CoV-2 藥物
4. **學術影響力高**：2006 年首發至今累計被引用數千次

#### 可改善之處

1. **再現性問題**：研究顯示跨版本成功率僅 17%
2. **學習曲線高**：需理解基因表達分析
3. **無證據整合**：僅提供預測排名，無臨床試驗驗證

## 競品 L：DeepDR Web Server

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://deepdr.xxx (arxiv 預印本) |
| 定位 | 整合式深度學習藥物再利用平台 |
| 發表 | arXiv 2511.08921 (2025) |
| 特色 | **第一個整合多種深度學習模型**的 Web 平台 |

#### 整合模型

DeepDR 整合 6 種深度學習工具：

| 模型 | 用途 |
|------|------|
| DeepDR | 疾病特定藥物再利用 |
| HetDR | 異質網路預測 |
| DeepDTnet | 藥物-靶點交互作用 |
| AOPEDF | 優化預測框架 |
| Cov-KGE | COVID-19 知識圖譜嵌入 |
| KG-MTL | 知識圖譜多任務學習 |

#### 資料整合

- **15+ 網路** + **5.9M 邊**（涵蓋 107 種關係類型）
- 連接藥物、疾病、蛋白質/基因、通路、表達
- 整合 6 個資料庫 + 2400 萬篇 PubMed 文獻

#### 優勢與特色

1. **免費開放**：無需註冊

3. **多模型選擇**：使用者可選擇不同深度學習模型
