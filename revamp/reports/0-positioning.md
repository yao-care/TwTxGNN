| 項目 | 內容 |

## 2. 目標受眾

### 主要受眾 (Primary)

| 項目 | 描述 |
|------|------|
| **是誰** | 醫藥研究人員、學術機構研究員 |
| **目標/需求** | 尋找有潛力的老藥新用研究方向，快速評估某藥物-疾病配對的現有證據 |
| **痛點** | 傳統文獻搜尋耗時、AI 預測工具只給分數沒給證據、難以判斷研究價值 |
| **使用場景** | 規劃研究計畫、撰寫研究提案、評估臨床試驗可行性 |

### 次要受眾 (Secondary)

| 項目 | 描述 |
|------|------|
| **是誰** | 藥廠研發人員、臨床醫師 |
| **目標/需求** | 評估藥物再利用商業價值、了解特定藥物的適應症擴展可能性 |
| **痛點** | 需要快速篩選大量候選、缺乏結構化的證據整理 |
| **使用場景** | 藥物資產評估、適應症擴展規劃、臨床決策參考 |

### 第三受眾 (Tertiary)

| 項目 | 描述 |
|------|------|
| **是誰** | 醫療 AI 研究者、生物資訊學家 |
| **目標/需求** | 了解 TxGNN 模型的實際應用案例、學習證據驗證方法論 |
| **痛點** | AI 模型論文多、實際應用案例少 |
| **使用場景** | 學術研究、方法論參考、教學 |

## 3. 差異化定位

| 項目 | 內容 |
|------|------|
| **與競品的差異** | 不只是 AI 預測分數，而是**預測 + 證據驗證**的完整報告。市面上的 drug repurposing 工具多半只提供預測結果，缺乏臨床證據整理 |
| **獨特優勢** | 1. 自動收集 ClinicalTrials.gov、PubMed、DrugBank 等多來源證據<br>2. 五級證據分類系統（L1-L5）<br>3. 可行動的決策建議（Go/Proceed/Consider/Explore/Hold）<br>4. 聚焦台灣健保藥品，具有在地實用性 |
| **不做什麼** | 不提供醫療建議、不預測藥物安全性、不替代臨床試驗、不進行藥物銷售 |

## 8. 現有問題觀察

基於首頁和方法論頁面的初步觀察：

| 問題 | 說明 | 優先級 |
|------|------|--------|
| Kramdown 語法殘留 | 頁面使用 `{: .fs-9 }` 等語法，可能在某些情況下顯示原始碼 | P0 |
| 缺少 Schema 標記 | 尚未看到 JSON-LD 結構化資料 | P0 |
| Meta 標籤不完整 | 需確認 OG/Twitter Card 是否完整 | P1 |
| 免責聲明不夠醒目 | YMYL 內容需要更明確的免責聲明 | P1 |
| 缺少作者資訊 | E-E-A-T 需要明確的作者/機構背書 | P1 |
