## 4. 機會識別

### 4.1 必須補的內容

| 內容 | 為什麼必須 | 競品參考 |
|------|------------|----------|
| sitemap.xml + robots.txt | SEO 基礎，無此則難被索引 | 所有競品都有 |
| Meta Description | 搜尋結果無摘要，點擊率低 | 基本 SEO 標準 |
| OG Tags | 社群分享無預覽圖 | 基本社群標準 |
| 作者/機構資訊 | E-E-A-T 必要，YMYL 內容更重要 | Open Targets |

### 4.2 差異化機會

| 機會 | 為什麼是機會 | 如何執行 |
|------|--------------|----------|
| **繁體中文關鍵字 SEO** | 競品全為英文，中文 drug repurposing 搜尋無競爭者 | 優化「老藥新用」「藥物再利用」等關鍵字 |
| **證據整合完整報告** | 競品只有預測分數，無臨床證據彙整 | 強調「預測 + 證據」的差異化 |
| **台灣藥品聚焦** | 無競品整合 TFDA 資料 | 突出「台灣健保藥品」定位 |
| **結構化證據等級** | L1-L5 系統獨特 | 加入 Schema 標記，爭取 Rich Snippets |

### 4.3 強化重點

| 現有優勢 | 如何強化 | 預期效果 |
|----------|----------|----------|
| 191 份完整報告 | 加入結構化資料、內部連結 | 提升索引和停留時間 |
| L1-L5 證據等級 | 視覺化、加入 Schema | 提升理解度和搜尋呈現 |

## 5. 關鍵發現摘要

### 最重要的 3 個發現

1. **SEO 基礎嚴重缺失**
   - 無 sitemap、robots.txt、meta description、OG tags
   - 197 頁內容無法被有效索引和分享
   - → **P0 優先修復，否則其他改善都無意義**

2. **E-E-A-T 信號不足**
   - 缺少作者資訊、機構說明、專家背書
   - 作為 YMYL（健康/醫療）內容，這是 Google 排名的關鍵因素
   - → **加入「關於我們」頁面、作者資訊、引用原始論文**

3. **差異化優勢未被強調**
   - 「預測 + 證據」是唯一差異化，但首頁強調不足
   - 台灣健保藥品聚焦是唯一中文資源，SEO 機會未把握
   - → **調整首頁文案、優化中文關鍵字**

### 改版重點方向

基於以上分析，建議改版聚焦於：

1. **SEO 基礎建設**：sitemap、robots.txt、meta description、OG tags
2. **E-E-A-T 強化**：作者資訊、機構說明、引用格式
3. **差異化強調**：調整首頁文案、突出「預測 + 證據」
4. **內部連結優化**：相關藥物推薦、麵包屑導航
5. **結構化資料**：Article、BreadcrumbList、FAQPage Schema
