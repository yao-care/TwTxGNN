## OG Image 規格

| 項目 | 規格 |
|------|------|
| 尺寸 | 1200 x 630 px |
| 格式 | PNG 或 JPG |
| 檔案大小 | < 1 MB |
| 路徑 | /assets/images/og-default.png |

### 設計要求

- 包含網站名稱「TwTxGNN」
- 包含標語「老藥新用，從數據到證據」
- 簡潔、專業風格
- 背景純色或簡單圖案
