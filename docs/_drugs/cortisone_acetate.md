# Cortisone Acetate：從過敏性炎症疾患到圓禿

## 一句話總結

Cortisone Acetate 是一種皮質固醇（類固醇），原本廣泛用於風濕性關節炎、過敏性疾患、支氣管氣喘及各類炎症性皮膚病。
TxGNN 模型預測它可能對**圓禿 (Alopecia Areata)** 有效，
目前有 **0 個臨床試驗**和 **20 篇文獻**（以 1950 年代案例系列為主）支持這個方向。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料（DrugBank MOA 為 Data Gap）。根據已知資訊，Cortisone Acetate 是一種皮質固醇（glucocorticoid），透過結合細胞內皮質固醇受體，廣泛抑制炎症因子的基因轉錄，發揮抗炎與免疫調節作用。

圓禿（Alopecia Areata）是一種自體免疫性疾病，主要機轉為 CD4⁺/CD8⁺ T 淋巴球對毛囊的攻擊，破壞毛囊周圍的免疫赦免（immune privilege），導致毛髮脫落。皮質固醇透過抑制此一自體免疫炎症反應，可減少毛囊破壞，理論上有助於毛髮再生。這一機轉關聯與 Cortisone Acetate 原有適應症（如風濕性關節炎、過敏性疾患等自體免疫/炎症性疾病）高度一致。

1950 年代多項臨床案例系列研究已直接記錄 Cortisone Acetate 用於圓禿的療效，包括針對重症圓禿全禿（alopecia totalis）及普禿（alopecia universalis）的口服或肌注治療。然而，現代皮質固醇治療圓禿已演進至局部注射三安諾（triamcinolone acetonide）等更精準療法，全身性使用 Cortisone Acetate 的長期安全性與療效需重新評估。

## 結論與下一步

**決策：Hold**

**理由：**
現有文獻證據全數來自 1950 至 1960 年代的案例系列（Tier 3），缺乏現代隨機對照試驗，無法確立全身性 Cortisone Acetate 對圓禿的療效與安全性優勢；現行圓禿治療標準已演進至局部注射皮質固醇或 JAK 抑制劑，全身性使用舊型皮質固醇的臨床定位尚不明確。

**若要推進需要：**
- 補充 DrugBank MOA 資料，確認作用機轉細節與現代皮質固醇的差異
- 評估與現行標準療法（triamcinolone 局部注射、baricitinib 等 JAK 抑制劑）的差異定位
- 設計現代化隨機對照試驗以取得 L1/L2 等級證據
- 建立全身性皮質固醇長期使用的安全性監測計畫（HPA 軸抑制、骨質疏鬆、感染風險等）
- 考慮特定次族群（如重症全禿、合併其他自體免疫疾病患者）是否有較高療效預測性
