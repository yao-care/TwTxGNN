# Threonine：從營養補充劑到胃輕癱的潛在應用

## 一句話總結

Threonine 原本用於營養補充及氨基酸補給。
TxGNN 模型預測它可能對 **Gastroparesis（胃輕癱）** 有效，
目前有 **1 篇文獻**提供間接支持，但缺乏直接臨床試驗證據。

## 快速總覽

| 項目 | 內容 |
|------|------|
| 原適應症 | 營養補充劑、氨基酸補給劑 |
| 預測新適應症 | Gastroparesis（胃輕癱） |

| 建議決策 | Hold |

## 為什麼這個預測合理？

Threonine（蘇氨酸）是一種必需氨基酸，參與蛋白質合成及多種代謝途徑。

**潛在機轉推測：**
1. **PI3K-AKT-mTOR 訊號通路**：相關文獻指出，糖尿病胃輕癱與胃平滑肌細胞凋亡及 PI3K-AKT-mTOR 訊號通路異常有關。氨基酸（包括 Threonine）可能透過調節 mTOR 訊號來影響胃平滑肌功能。
2. **蛋白質代謝支持**：Threonine 作為必需氨基酸，可能有助於維持或修復胃平滑肌細胞。
3. **腸黏膜完整性**：Threonine 是腸道黏蛋白合成的重要成分，可能對胃腸道功能有保護作用。

然而，目前缺乏直接證據支持 Threonine 對胃輕癱的療效。

## 臨床試驗證據

目前無 Threonine 與 Gastroparesis 相關的臨床試驗登記。

## 安全性考量

- **藥物交互作用**：未發現顯著藥物交互作用記錄
- **一般安全性**：作為必需氨基酸，在正常劑量下安全性良好
- **特殊族群**：
  - 孕婦及哺乳期婦女：依複方製劑仿單規定
  - 腎功能不全患者：需監測氨基酸代謝

**注意事項：**
- 現有許可證主要用於營養補充，非治療胃輕癱
- 靜脈營養製劑需在醫療監督下使用

## 結論與下一步

**決策：Hold**

**理由：**
1. 目前僅有間接機轉相關文獻，無直接療效證據
2. 缺乏針對胃輕癱的臨床試驗數據
3. Threonine 對胃輕癱的具體作用機轉尚未明確

**若要推進需要：**
- 動物模型驗證 Threonine 對胃輕癱的直接療效
- 探索最佳給藥途徑（口服 vs. 靜脈注射）
- 設計 Phase I/II 臨床試驗評估安全性與初步療效
- 確認劑量範圍及給藥方案
