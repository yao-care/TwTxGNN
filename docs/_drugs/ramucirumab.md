# Ramucirumab：從胃癌到子宮韌帶腺癌

## 一句話總結

Ramucirumab 是一種 VEGFR-2 抑制劑，原本用於治療胃癌、非小細胞肺癌、大腸直腸癌及肝細胞癌。
TxGNN 模型預測它可能對**子宮韌帶腺癌 (Uterine Ligament Adenocarcinoma)** 有效，
但目前沒有臨床試驗或文獻支持這個方向。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據已知資訊，Ramucirumab 是 VEGFR-2 抑制劑，
其成分在胃癌、非小細胞肺癌、大腸直腸癌及肝細胞癌中的療效已被證實，
機轉上可能適用於子宮韌帶腺癌，因為它可能通過抑制腫瘤血管生成來影響腺癌的生長。

**若要推進需要：**
