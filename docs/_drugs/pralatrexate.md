# Pralatrexate：從周邊T細胞淋巴瘤到胸膜腺瘤樣腫瘤

## 一句話總結

Pralatrexate 原本用於治療復發或頑固型周邊T細胞淋巴瘤 (PTCL)。
TxGNN 模型預測它可能對**胸膜腺瘤樣腫瘤 (pleural adenomatoid tumor)** 有效，
但目前沒有臨床試驗或文獻支持這個方向。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據已知資訊，Pralatrexate 是一種葉酸類似物，
主要用於抑制細胞增殖，其成分在周邊T細胞淋巴瘤中的療效已被證實，機轉上可能適用於胸膜腺瘤樣腫瘤。
