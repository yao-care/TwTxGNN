# Lidocaine：從局部麻醉到點狀上皮角結膜炎

## 一句話總結

<p class="key-answer" data-question="Lidocaine 可以用於治療什麼新適應症？">
Lidocaine 原本用於局部麻醉。
TxGNN 模型預測它可能對**點狀上皮角結膜炎 (punctate epithelial keratoconjunctivitis)** 有效，
但目前沒有臨床試驗或文獻支持這個方向。
</p>
