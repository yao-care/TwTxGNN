# Aluminum Hydroxide：從胃酸過多到活動性消化性潰瘍

## 一句話總結

<p class="key-answer" data-question="Aluminum Hydroxide 可以用於治療什麼新適應症？">
Aluminum hydroxide 原本用於緩解胃酸過多及胃部不適。

## 為什麼這個預測合理？

Aluminum hydroxide 是經典的制酸劑，透過中和胃酸來保護胃黏膜。其作用機轉包括：

1. **中和胃酸**：直接中和已分泌的胃酸，提升胃內 pH 值
2. **抑制胃蛋白酶活性**：在較高 pH 環境下，胃蛋白酶活性顯著降低
3. **黏膜保護**：研究顯示含鋁制酸劑具有胃黏膜保護作用

從原適應症（胃酸過多、胃炎）到預測適應症（活動性消化性潰瘍），
機轉上具有合理連結，皆涉及胃酸中和與黏膜保護。

## 安全性考量

- **藥物交互作用**：
  - **重大交互作用**：與 Dolutegravir（抗HIV藥物）併用會降低其吸收
  - **中度交互作用**：與 Acalabrutinib 併用需注意
  - **輕微交互作用**：與 Famotidine、Ranitidine、Acebutolol 等
- **長期使用注意事項**：長期大量使用可能導致鋁蓄積，尤其在腎功能不全患者

## 結論與下一步

**決策：Proceed with Caution**

**理由：**
Aluminum hydroxide 作為制酸劑用於消化性潰瘍有歷史文獻支持，但現代治療已有更有效的選擇（如 PPI、H2 阻斷劑）。預測適應症與原適應症重疊度高，可視為原適應症的延伸而非真正的老藥新用。

**若要推進需要：**
- 與現代標準治療的比較性研究
- 明確的給藥劑量與療程建議
- 長期安全性（特別是鋁蓄積）的評估
