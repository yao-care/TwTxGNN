# Nitroxoline（耐挫索林）：從泌尿道感染到閉經 (Amenorrhea)

## 一句話總結

Nitroxoline 是一種 8-羥基喹啉衍生物抗菌劑，原本用於尿道炎、膀胱炎、腎盂腎炎等泌尿道感染症的治療。
TxGNN 模型預測它可能對**閉經 (Amenorrhea)** 有效，
目前**尚無臨床試驗**和**文獻**直接支持這個方向，預測依據僅來自知識圖譜模型。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據現有文獻線索，Nitroxoline 是一種 8-羥基喹啉衍生物，其主要藥理機轉被認為涉及**螯合細菌所需的鋅／鐵離子**，從而抑制細菌代謝酶的活性，達到抑菌效果。在泌尿道感染中的療效已有數十年臨床使用紀錄。

然而，閉經（Amenorrhea）的病理生理機轉主要涉及下視丘－腦垂腺－卵巢軸的荷爾蒙調控異常，與 Nitroxoline 已知的金屬離子螯合抗菌機轉**目前無已知交集**。無論是原發性或繼發性閉經，其病因學上均與抗菌藥物的作用途徑相去甚遠。

本預測的唯一依據為 TxGNN 知識圖譜模型的推斷。在無任何臨床試驗或基礎研究佐證的情況下，此預測的生物學合理性尚待釐清，建議暫不推進。

## 結論與下一步

**決策：Hold**

**理由：**
TxGNN 模型雖給出高預測分數（99.74%），但閉經的發病機轉（荷爾蒙軸失調）與 Nitroxoline 的抗菌機轉（金屬離子螯合）在生物學上缺乏合理連結，且目前完全沒有任何臨床試驗或文獻支持此適應症方向，證據等級僅為 L5。

**若要推進需要：**
- 補充 Nitroxoline 的完整作用機轉資料（DrugBank API 查詢，數據缺口 DG001）
- 釐清 TxGNN 預測此適應症的圖譜路徑依據（知識圖譜中間節點分析）
- 評估 Nitroxoline 是否有已知的荷爾蒙相關生物活性（文獻系統性回顧）
- 如發現合理機轉連結，再啟動前臨床體外研究設計
