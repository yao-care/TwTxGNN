# Camphor：從心臟衰弱到偏頭痛

## 一句話總結

Camphor 是一種天然萜類化合物，台灣許可適應症以心臟衰弱之興奮、呼吸困難及新生兒窒息的急性支持為代表。
TxGNN 模型預測它可能對**偏頭痛 (Migraine Disorder)** 有效，
目前有 **0 個臨床試驗**和 **5 篇文獻**支持這個方向。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據已知資訊，Camphor 是一種廣泛用於外用止痛、皮膚刺激及急性心肺支持的天然化合物，其藥理活性已在零星研究中被探討，機轉上存在與偏頭痛的理論連結。

Camphor 被認為可作為 TRPV1（瞬態受體電位香草素受體）激動劑，持續活化後可使疼痛受體脫敏；同時透過 TRPM8 冷覺受體產生感覺抑制效果。理論上，這一特性可能干預三叉神經血管疼痛訊號，而三叉神經血管系統的過度活化正是偏頭痛發作的核心機轉之一。

然而，現有文獻不僅缺乏直接支持，部分個案報告甚至顯示含 camphor 精油**可能誘發或加重**叢發性頭痛，機轉推論的臨床支持強度相當有限，須謹慎解讀預測結果。

## 結論與下一步

**決策：Hold**

**理由：**
Camphor 對 TRPV1/TRPM8 受體的調節雖在機轉上提供了偏頭痛的理論基礎，但現有個案報告顯示含 camphor 精油可能**引發而非緩解**頭痛，在缺乏任何前瞻性臨床設計的情況下，不建議推進此再利用方向。

**若要推進需要：**
- 補充 Camphor 詳細作用機轉資料（特別是對三叉神經血管系統的影響）
- 釐清現有個案報告中 camphor 引發頭痛的劑量、劑型與給藥途徑，排除安全性疑慮
- 設計針對外用或吸入給藥途徑的小型 Proof-of-concept 研究（需與神經科/頭痛專科合作）
- 建立神經毒性風險評估計畫（camphor 中毒可引發驚厥，高劑量安全性需事先確認）
