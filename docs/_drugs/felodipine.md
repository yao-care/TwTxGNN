## 為什麼這個預測合理

Felodipine 作為二氫吡啶類鈣離子通道阻斷劑，其對預測適應症的潛在療效有以下藥理學基礎：

1. **血管擴張作用**：阻斷 L-type 鈣離子通道，導致動脈平滑肌鬆弛
2. **肺血管擴張**：理論上可降低肺血管阻力
3. **腎血管效應**：可改善腎臟灌流，對惡性高血壓性腎病可能有保護作用
4. **同類藥物經驗**：其他鈣離子通道阻斷劑(如 nifedipine)已被用於某些類型的肺高壓

### 限制因素
- Felodipine 對周邊血管的選擇性高於肺血管
- 現代肺高壓治療主要使用專門的肺血管擴張劑(如 sildenafil, bosentan)
- 鈣離子通道阻斷劑僅對約 10% 的肺高壓患者有效

## 臨床試驗證據

### ClinicalTrials.gov 搜尋結果

**未發現** Felodipine 用於以下預測適應症的臨床試驗：
- 肺高壓(與肺疾病/缺氧相關)
- 惡性腎血管性高血壓
- 惡性高血壓性腎病

**證據等級：L5 (僅預測)**

## 安全性考量

### 已知風險
- **周邊水腫**：最常見副作用，與劑量相關
- **頭痛**：血管擴張相關
- **潮紅**：約 10% 患者
- **心悸**：反射性心跳加速
- **牙齦增生**：長期使用可能發生

### 藥物交互作用

| 交互作用藥物 | 機轉 | 臨床意義 |
|-------------|------|---------|
| CYP3A4 抑制劑 | 代謝減慢 | 血壓過度降低 |
| 葡萄柚汁 | CYP3A4 抑制 | 避免併用 |
| Beta-blockers | 協同降壓 | 可能需要減量 |
| Digoxin | 血中濃度升高 | 監測 digoxin 濃度 |

### 特殊族群
- **孕婦**：C 級，不建議使用
- **哺乳**：分泌至乳汁，不建議使用
- **肝功能不全**：需減量
- **老年人**：可能需較低起始劑量

### 肺高壓特殊考量
如用於肺高壓，需注意：
- 僅對血管反應性陽性的患者有效
- 需在專業中心進行急性血管反應性試驗
- 可能導致全身性低血壓而惡化右心功能

## 結論與下一步

### 整體評估
此預測**不建議直接臨床應用**，原因如下：
1. 缺乏專門的臨床試驗證據
2. 現有更有效的肺高壓專用藥物
3. Felodipine 對肺血管的選擇性不如其他藥物
4. 惡性高血壓通常需要更積極的介入治療

### 建議行動
- [ ] 對於惡性高血壓，可使用其他更快速作用的降壓藥
- [ ] 肺高壓應由專科醫師使用核准的專門藥物治療
- [ ] 不建議將 felodipine 作為這些適應症的一線選擇

### 特殊情況考量
在無法取得專門肺高壓藥物的緊急情況下，鈣離子通道阻斷劑可能是一個備選，但需要密切監測。

### 風險等級
**高風險** - 不建議超適應症使用於肺高壓；惡性高血壓需要更積極的治療策略
