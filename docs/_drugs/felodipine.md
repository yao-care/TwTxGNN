<h3>為什麼這個預測合理？</h3>

<p>Felodipine 作為二氫吡啶類鈣離子通道阻斷劑，其對預測適應症的潛在療效有以下藥理學基礎：</p>

<ol>
<li><strong>血管擴張作用</strong>：阻斷 L-type 鈣離子通道，導致動脈平滑肌鬆弛</li>
<li><strong>肺血管擴張</strong>：理論上可降低肺血管阻力</li>
<li><strong>腎血管效應</strong>：可改善腎臟灌流，對惡性高血壓性腎病可能有保護作用</li>
<li><strong>同類藥物經驗</strong>：其他鈣離子通道阻斷劑(如 nifedipine)已被用於某些類型的肺高壓</li>
</ol>

<h3>臨床試驗</h3>

<p>### ClinicalTrials.gov 搜尋結果</p>

<p><strong>未發現</strong> Felodipine 用於以下預測適應症的臨床試驗：</p>
<ul>
<li>肺高壓(與肺疾病/缺氧相關)</li>
<li>惡性腎血管性高血壓</li>
<li>惡性高血壓性腎病</li>

</ul>
<p><strong>證據等級：L5 (僅預測)</strong></p>

- 含 Felodipine + Enalapril 的複方製劑用於高血壓治療
- 不建議作為起始治療

## 安全性考量

### 已知風險
- **周邊水腫**：最常見副作用，與劑量相關
- **頭痛**：血管擴張相關
- **潮紅**：約 10% 患者
- **心悸**：反射性心跳加速
- **牙齦增生**：長期使用可能發生

### 藥物交互作用

| 交互作用藥物 | 機轉 | 臨床意義 |
|-------------|------|---------|
| CYP3A4 抑制劑 | 代謝減慢 | 血壓過度降低 |
| 葡萄柚汁 | CYP3A4 抑制 | 避免併用 |
| Beta-blockers | 協同降壓 | 可能需要減量 |
| Digoxin | 血中濃度升高 | 監測 digoxin 濃度 |

### 特殊族群
- **孕婦**：C 級，不建議使用
- **哺乳**：分泌至乳汁，不建議使用
- **肝功能不全**：需減量
- **老年人**：可能需較低起始劑量

### 肺高壓特殊考量
如用於肺高壓，需注意：
- 僅對血管反應性陽性的患者有效
- 需在專業中心進行急性血管反應性試驗
- 可能導致全身性低血壓而惡化右心功能




### 藥物-食物交互作用 (DFI)

<div class="dfi-source">資料來源：<a href="https://ddinter2.scbdd.com/" target="_blank">DDInter 2.0</a>（原文內容請參閱該網站）</div>

**酒精** 🟡 Moderate
- 影響：產生協同作用。
- 建議：建議避免併用。需監測療效或不良反應。

**葡萄柚汁** 🟡 Moderate
- 影響：影響藥物代謝。可能增加藥物血中濃度。
- 建議：避免大量攝取。需監測療效或不良反應。避免食用葡萄柚或葡萄柚汁。

### 藥物-草藥交互作用 (DHI)

**聖約翰草（貫葉連翹）** 🔴 Major
- 影響：聖約翰草降低鈣離子通道阻斷劑療效
- 建議：禁止併用


### 藥物-疾病注意事項 (DDSI)

<div class="ddsi-source">資料來源：<a href="https://ddinter2.scbdd.com/" target="_blank">DDInter 2.0</a>（原文內容請參閱該網站）</div>

**心臟衰竭** 🟡 Moderate
- 應謹慎使用本藥物。可能有嚴重不良反應。

**Shock, Cardiogenic** 🟢 Minor
- 本藥物在此情況下禁用。風險包括：低血壓。

**Coronary Artery Disease** 🟢 Minor
- 可能有嚴重不良反應。

**肝臟疾病** 🟢 Minor
- 需定期監測。可能有嚴重不良反應。

## 結論與下一步

### 整體評估
此預測**不建議直接臨床應用**，原因如下：
1. 缺乏專門的臨床試驗證據
2. 現有更有效的肺高壓專用藥物
3. Felodipine 對肺血管的選擇性不如其他藥物
4. 惡性高血壓通常需要更積極的介入治療

### 建議行動
- [ ] 對於惡性高血壓，可使用其他更快速作用的降壓藥
- [ ] 肺高壓應由專科醫師使用核准的專門藥物治療
- [ ] 不建議將 felodipine 作為這些適應症的一線選擇

### 特殊情況考量
在無法取得專門肺高壓藥物的緊急情況下，鈣離子通道阻斷劑可能是一個備選，但需要密切監測。

### 風險等級
**高風險** - 不建議超適應症使用於肺高壓；惡性高血壓需要更積極的治療策略
