# Amcinonide：從異位性皮膚炎到皮膚炎

## 一句話總結

Amcinonide 原本用於治療急慢性皮膚病如異位性皮膚炎、接觸性皮膚炎、牛皮癬等。
TxGNN 模型預測它可能對**皮膚炎 (dermatitis)** 有效，
目前有 **20 篇文獻**支持這個方向。

## 為什麼這個預測合理？

Amcinonide 是高效價局部類固醇，屬於副腎皮質荷爾蒙類藥物。其作用機轉包括：

1. **抗發炎作用**：抑制發炎介質的釋放，減少血管擴張與水腫
2. **抗過敏作用**：穩定肥大細胞，減少組織胺釋放
3. **免疫調節**：抑制免疫細胞的活化與遷移

預測適應症「皮膚炎」實際上已包含在原適應症範圍內（如異位性皮膚炎、接觸性皮膚炎皆屬皮膚炎），此預測印證了模型對該藥物適應症的正確識別。

## 安全性考量

- **藥物交互作用**：
  - **輕微交互作用**：與多種降血糖藥物（Metformin、Glimepiride、各類胰島素等）併用可能影響血糖控制
- **局部使用注意事項**：
  - 長期使用可能導致皮膚萎縮、毛細血管擴張
  - 大面積或封閉式使用可能增加全身性吸收風險
- **過敏反應**：文獻報告顯示部分患者對 amcinonide 產生接觸性皮膚炎，需注意交叉過敏

## 結論與下一步

**決策：Already Approved (原適應症範圍內)**

**理由：**
預測適應症「皮膚炎」實際上已包含在原核准適應症中。Amcinonide 作為高效價局部類固醇，對各類皮膚炎均有療效。此預測驗證了模型對藥物-疾病關係的正確識別。

**臨床建議：**
- 依據皮膚炎嚴重程度選擇適當效價的類固醇
- 注意使用部位（臉部、皺褶處應避免使用高效價類固醇）
- 監測長期使用的局部副作用
