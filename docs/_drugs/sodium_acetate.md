# Sodium Acetate：從血液透析到先天性凝血酶原缺乏症

## 一句話總結

Sodium Acetate（乙酸鈉）是一種電解質補充劑，廣泛用於血液透析、靜脈營養液調配及代謝性酸中毒的矯正。
TxGNN 模型預測它可能對**先天性凝血酶原缺乏症 (congenital prothrombin deficiency)** 有效，
目前**無任何臨床試驗或文獻**支持這個方向，屬純模型預測層級。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據已知資訊，Sodium Acetate 是一種無機鹽類電解質，主要作為鹼化劑與鈉離子補充來源，在靜脈營養調配、血液透析及電解質失衡的臨床情境中被廣泛使用。

先天性凝血酶原缺乏症（Factor II 缺乏）是一種罕見的遺傳性凝血障礙，患者體內凝血酶原的合成或功能表現異常。就理論機轉而言，Sodium Acetate 代謝後可升高體液 pH，理論上可透過酸鹼正常化間接維持肝臟凝血因子合成所需的代謝環境；然而，目前完全缺乏任何直接機轉證據或臨床數據，將醋酸鈉與凝血酶原（Factor II）的表現或功能調控加以連結。

整體而言，TxGNN 的高分預測可能源於知識圖譜中「電解質補充 → 代謝環境正常化 → 肝臟合成路徑」的遠程節點關聯，但機轉鏈結薄弱，生物學合理性極為有限，不足以支撐直接的再利用假說。

## 結論與下一步

**決策：Hold**

**理由：**
TxGNN 預測分數雖高（99.98%），但完全缺乏針對先天性凝血酶原缺乏症的臨床試驗、文獻證據及合理機轉連結，目前屬 L5 純模型預測層級。在無任何前臨床支持的情況下，不建議進一步資源投入。
