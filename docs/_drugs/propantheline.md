# Propantheline：從消化性潰瘍到胃十二指腸炎

## 一句話總結

Propantheline 是一種副交感神經阻斷劑，原本用於胃潰瘍、十二指腸潰瘍和胃炎的治療。TxGNN 模型預測它可能對**胃十二指腸炎 (Gastroduodenitis)** 有效，目前有 **5 篇文獻**支持這個方向。

| 許可證數 | 多張 |
| 建議決策 | Go |

## 為什麼這個預測合理？

Propantheline 是一種抗膽鹼藥物，能夠阻斷副交感神經的作用，減少胃酸分泌並降低胃腸道的蠕動和痙攣。這些作用機轉使其成為治療消化性潰瘍和胃炎的有效藥物。

胃十二指腸炎是指胃和十二指腸同時發生的炎症，其病理機轉與胃潰瘍和十二指腸潰瘍高度相關。Propantheline 透過以下機轉可能有助於治療胃十二指腸炎：

1. 抑制胃酸分泌，減少酸性環境對黏膜的刺激
2. 減少胃腸道蠕動，降低機械性損傷
3. 緩解胃腸痙攣，減輕疼痛症狀

由於原適應症和新預測適應症在解剖位置和病理機轉上高度相似，這個預測具有很強的合理性。

## 安全性考量

安全性資訊請參考原廠仿單。

抗膽鹼藥物的常見注意事項：
- 口乾
- 視力模糊
- 尿滯留（特別是攝護腺肥大患者）
- 便秘

## 結論與下一步

**決策：Go**

**理由：**
胃十二指腸炎與 propantheline 的原適應症（胃潰瘍、十二指腸潰瘍、胃炎）在病理機轉上高度相關。Propantheline 的抗膽鹼作用（抑制胃酸分泌、減少胃腸蠕動）對於胃十二指腸炎的治療具有明確的藥理學基礎。文獻中已有 propantheline 用於胃十二指腸潰瘍治療的臨床研究。

**若要推進需要：**
- 確認台灣目前是否有有效的許可證（部分已註銷）
- 評估與現代質子泵抑制劑（PPI）相比的定位
- 制定適當的劑量方案
