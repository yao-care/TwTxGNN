# Mexiletine：從心室性不整律到多毛症

## 一句話總結

Mexiletine（鹽酸美克律定）是第 IB 類抗心律不整藥，在台灣核准用於心室性不整律的治療或預防。
TxGNN 模型預測它可能對**多毛症 (Hypertrichosis)** 有效，預測分數達 **99.78%**，
但目前**無任何臨床試驗或文獻**支持這個方向，此預測可能源於知識圖譜拓撲相似性而非直接藥理關聯。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料（已列為資料缺口 DG001）。根據 Evidence Pack 各適應症的機轉描述可知，Mexiletine 屬於**第 IB 類電壓門控鈉離子通道（Nav）阻斷劑**，透過穩定神經膜及心肌膜的靜息電位、抑制異常的鈉離子內流，降低心室異位放電，達到控制心律不整的效果。

多毛症（Hypertrichosis）是一種毛髮過度生長的疾病，病理機轉主要涉及毛囊生長調控路徑（如 Wnt/β-catenin、FGF 訊號等）。Mexiletine 的鈉離子通道阻斷作用與上述毛囊調控路徑**缺乏已知的直接藥理關聯**，原適應症（心律不整）與多毛症在生理系統上亦無明顯共通性。

此預測最可能源於**知識圖譜拓撲相似性**——即在 TxGNN 知識圖譜中，Mexiletine 節點與多毛症節點在圖結構上距離較近，而非藥理機轉的直接支持。若要推進此方向，需要前臨床研究先驗證假說。

## 結論與下一步

**決策：Hold**

**理由：**
Mexiletine 的鈉離子通道阻斷機轉與多毛症的毛囊生長調控路徑（Wnt/β-catenin、FGF 等）缺乏已知藥理關聯，且完全無任何臨床試驗或文獻支持，目前僅有模型預測分數，證據基礎不足以支持推進。

**若要推進需要：**
- 補齊 MOA 資料缺口（DG001）：查詢 DrugBank API 取得完整作用機轉描述
- 建立可驗證的藥理假說，說明 Nav 通道抑制如何影響毛囊生長路徑
- 執行毛囊細胞 in vitro 實驗（如 dermal papilla cell 模型），驗證 Mexiletine 是否具有生物活性
- 評估 106 筆高 DDI 負擔對潛在應用情境的臨床可行性
