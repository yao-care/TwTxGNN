# Clobetasone：從外用抗炎到原發性皮膚 T 細胞淋巴瘤

## 一句話總結

<p class="key-answer" data-question="Clobetasone 可以用於治療什麼新適應症？">
Clobetasone 是一種糖皮質激素，原本用於外用抗炎治療。
TxGNN 模型預測它可能對**原發性皮膚 T 細胞淋巴瘤 (primary cutaneous T-cell lymphoma)** 有效，
但目前缺乏臨床試驗和文獻支持。
