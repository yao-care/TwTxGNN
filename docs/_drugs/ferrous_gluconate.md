# Ferrous Gluconate (葡萄糖酸亞鐵) - 藥師評估報告

## 一句話總結

葡萄糖酸亞鐵是一種傳統補鐵劑，TxGNN 預測其可能對 Plummer-Vinson 症候群等罕見貧血相關疾病有療效，這些預測與其已知的鐵補充機制高度一致。

## 快速總覽

| 項目 | 內容 |
|------|------|
| 藥物學名 | Ferrous Gluconate |
| 台灣商品名 | 葡萄糖酸鐵糖衣錠、維他葡萄糖鐵糖衣錠、血命健糖衣片等 |
| DrugBank ID | DB14488 |
| 原核准適應症 | 鐵缺乏性貧血、妊娠貧血、營養不良 |
| 預測新適應症 | Plummer-Vinson 症候群、維生素缺乏症、巨紅血球性貧血 |
| 最高證據等級 | L3 (觀察性研究) |
| 台灣上市狀態 | 有效許可證 |

## 為什麼這個預測合理？

葡萄糖酸亞鐵作為鐵補充劑，其預測適應症與原核准適應症具有高度的機轉相似性：

1. **Plummer-Vinson 症候群** (TxGNN Score: 0.999, Rank: 1843)：此症候群的典型三聯症為缺鐵性貧血、吞嚥困難和食道網狀膜，鐵劑補充是標準治療方式，預測非常合理。

2. **維生素 B12 和葉酸獨立性先天性巨紅血球性貧血** (TxGNN Score: 0.999, Rank: 1894)：雖然名稱提示與 B12/葉酸無關，但鐵缺乏可能是此類貧血的共病因素。

3. **維生素缺乏症** (TxGNN Score: 0.992, Rank: 14603)：鐵劑常與維生素複合製劑併用，且鐵缺乏常與其他營養素缺乏同時出現。

### 一般注意事項
- 建議空腹服用以增加吸收
- 與維生素 C 併服可增加吸收
- 可能引起腸胃不適、便秘或黑便
- 過量可能導致鐵中毒

## 結論與下一步

### 預測評估結論

葡萄糖酸亞鐵的預測適應症（Plummer-Vinson 症候群、維生素缺乏相關貧血）與其作為鐵補充劑的藥理機轉高度一致。Plummer-Vinson 症候群的預測尤其具有臨床意義，因為鐵補充確實是該症候群的標準治療。

### 證據等級總結

| 預測適應症 | TxGNN Score | 證據等級 | 評估 |
|------------|-------------|----------|------|
| Plummer-Vinson 症候群 | 0.999 | L5 | 機轉合理，臨床已確立 |
| 巨紅血球性貧血 | 0.999 | L5 | 可能為共病治療 |
| 維生素缺乏症 | 0.992 | L3 | 有臨床試驗支持 |

### 建議

1. **Plummer-Vinson 症候群**：此預測實際上反映了既有的臨床實務，無需額外研究即可確認
2. **維生素缺乏症**：作為複合營養補充的一部分，臨床價值已確立
3. **其他預測適應症**：需更多證據支持
