# Amorolfine：從抗黴菌劑到藥物誘發性骨質疏鬆

## 一句話總結

Amorolfine 是一種抗黴菌劑，主要用於治療由真菌引起的感染。
TxGNN 模型預測它可能對**藥物誘發性骨質疏鬆 (Drug-Induced Osteoporosis)** 有效，
但目前沒有臨床試驗或文獻支持這個方向。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據已知資訊，Amorolfine 是抗黴菌劑的一部分，
其成分在抗黴菌治療中的療效已被證實，但機轉上與藥物誘發性骨質疏鬆的直接關聯性不明。

## 結論與下一步

**決策：Hold**

**理由：**
目前缺乏臨床試驗和文獻支持 Amorolfine 用於藥物誘發性骨質疏鬆的療效，且其作用機轉與骨質疏鬆的直接關聯性不明。

**若要推進需要：**
