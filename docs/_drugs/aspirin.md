# Aspirin：從鎮痛解熱到新適應症探索

## 一句話總結

Aspirin 主要用於解熱鎮痛及預防心血管疾病。
目前尚無新的適應症預測，但其廣泛的藥理作用提示可能有其他潛在用途。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據已知資訊，Aspirin 是一種常用的解熱鎮痛藥，
其成分在緩解疼痛和發炎反應中的療效已被證實，並且在心血管疾病的預防中也有應用。
儘管缺乏新適應症的預測，Aspirin 的廣泛作用機轉提示可能有其他潛在用途。
