# Voriconazole：抗黴菌藥物

## 一句話總結

Voriconazole 是一種廣效抗黴菌藥物，用於治療侵犯性麴菌症及其他嚴重黴菌感染。
TxGNN 模型**未預測**出任何新的適應症。

## 為什麼沒有預測新適應症？

Voriconazole 是第二代三唑類抗黴菌藥物，透過抑制真菌細胞色素 P450 依賴性 14α-去甲基酶，阻斷麥角固醇合成，破壞真菌細胞膜完整性。其作用機轉高度專一於真菌，對人類細胞影響小。由於其藥理作用針對真菌特有的代謝途徑，TxGNN 知識圖譜分析未發現與其他非感染性疾病的顯著關聯。

## 安全性考量

### 重大藥物交互作用（Major）

| 交互作用藥物 | 機轉說明 |
|-------------|---------|
| Triamcinolone | CYP3A4 抑制，增加類固醇暴露 |
| Budesonide | CYP3A4 抑制，增加類固醇暴露 |
| Loperamide | 增加 QT 延長風險 |
| Chlorpropamide | 增加低血糖風險 |
| Glimepiride | 增加低血糖風險 |
| Dolasetron | 增加 QT 延長風險 |
| Eliglustat | CYP3A4 抑制 |
| Naloxegol | 增加鴉片類戒斷症狀風險 |

### 中度藥物交互作用（Moderate）

| 交互作用藥物 | 機轉說明 |
|-------------|---------|
| 質子幫浦抑制劑（Omeprazole, Lansoprazole 等） | CYP2C19 競爭 |
| Dexamethasone | 相互影響代謝 |
| Clarithromycin | CYP3A4 抑制 |
| Ondansetron, Palonosetron | 潛在 QT 延長 |
| Pioglitazone, Saxagliptin | 增加降血糖藥物暴露 |
| Levofloxacin | QT 延長風險 |

### 重要安全警語

- **視覺障礙**：約 30% 患者可能出現可逆性視覺改變
- **肝毒性**：需監測肝功能
- **QT 延長**：避免與其他延長 QT 藥物併用
- **光敏感性**：長期使用增加皮膚癌風險
- **CYP2C19 多型性**：基因型影響藥物代謝

### 治療藥物監測

建議進行 Voriconazole 血中濃度監測：
- 目標谷濃度：1-5.5 mg/L
- 濃度過低：療效不足
- 濃度過高：增加毒性風險

## 結論與下一步

**決策：無老藥新用候選**

**理由：**
Voriconazole 的作用機轉高度專一於真菌細胞，TxGNN 模型未發現具有足夠證據支持的新適應症候選。該藥物在現有抗黴菌適應症中已展現優異療效。

**臨床建議：**
- 維持現有抗黴菌適應症的使用
- 注意藥物交互作用，特別是 CYP3A4 和 CYP2C19 相關藥物
- 建議進行治療藥物監測以優化療效和安全性
- 長期使用患者需監測視力和皮膚變化
