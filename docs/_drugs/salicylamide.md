# Salicylamide：從解熱鎮痛到感冒相關疾病

## 一句話總結

Salicylamide 原本用於解熱鎮痛。TxGNN 模型預測它可能對**普通感冒 (common cold)** 及**咽炎 (pharyngitis)** 有效，這些預測獲得多篇歷史文獻支持。

## 快速總覽

| 項目 | 內容 |
|------|------|
| 原適應症 | 解熱鎮痛、風濕關節痛、神經痛、肌肉痛 |
| 預測新適應症 | 普通感冒 (common cold)、咽炎 (pharyngitis) |

| 建議決策 | Worth Exploring |

## 為什麼這個預測合理？

Salicylamide 是水楊酸的醯胺衍生物，具有以下藥理特性：

1. **解熱作用**：抑制前列腺素合成，降低發熱反應。

2. **鎮痛作用**：緩解感冒相關的頭痛、咽喉痛等症狀。

3. **抗發炎作用**：減輕上呼吸道感染相關的發炎反應。

4. **複方製劑經驗**：歷史上常與其他成分組成感冒藥複方，有實際臨床使用經驗。

## 臨床試驗證據

目前無新的臨床試驗登記，但歷史上有多項臨床使用經驗。

### 咽炎相關文獻（3 篇）

   - 重點：兒科使用經驗，包含咽炎等上呼吸道感染

2. **Ricciardiello (1969)** - Minerva Medica

   - 類型：雙盲研究
   - 重點：含 Salicylamide 的製劑用於咽扁桃腺炎

3. **Guaschino (1953)** - Gazzetta Medica Italiana
   - 標題：Therapy of catarrhal tonsillitis in infancy with o-hydroxybenzamide (salicylamide) associated with sodium p-aminobenzoate
   - 重點：嬰幼兒卡他性扁桃腺炎的治療經驗
   - [PubMed 連結](https://pubmed.ncbi.nlm.nih.gov/13060598/)

## 安全性考量

### 藥物交互作用

目前 DDInter 資料庫中無 Salicylamide 的重大藥物交互作用記錄。

### 特殊注意事項

- **過量毒性**：有文獻報告過量使用導致毒性反應
- **肝代謝**：主要經葡萄糖醛酸化代謝，肝功能不全者需注意
- **類水楊酸反應**：可能發生類似阿斯匹靈的過敏反應

### 與阿斯匹靈的差異

根據文獻，阿斯匹靈過敏的患者可能可以安全使用 Salicylamide，因為其抑制環氧化酶的能力較弱。

## 結論與下一步

**決策：Worth Exploring**

**理由：**
1. 藥理機轉支持其用於感冒相關症狀
2. 有歷史臨床使用經驗和文獻支持
3. 台灣仍有有效許可證
4. 安全性特徵相對良好

**若要推進需要：**

- 與現有感冒藥物的比較性研究
- 考慮複方製劑的開發可能性
