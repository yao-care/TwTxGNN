# Potassium Iodide (碘化鉀) - 藥師評估報告

## 一句話總結

碘化鉀是一種具有祛痰、抗真菌及甲狀腺保護作用的藥物,TxGNN預測其可用於鼻腔疾病治療,已有文獻證據支持其對真菌性鼻竇炎的療效。

## 為什麼這個預測合理?

### 鼻腔疾病預測分析 (TxGNN Score: 0.9995, Rank: 1593)

1. **抗真菌活性**: 碘化鉀具有已知的抗真菌活性,特別是對孢子絲菌病和接合菌病
2. **黏膜作用**: 碘離子可促進黏液分泌和清除
3. **免疫調節**: 可能具有增強局部免疫反應的作用
4. **歷史應用**: 傳統上用於治療深部真菌感染

### 急性喉咽炎預測分析 (TxGNN Score: 0.9995, Rank: 1664)

1. **祛痰作用**: 碘化鉀可增加呼吸道分泌物,有助於痰液排出
2. **抗菌特性**: 碘具有廣譜抗菌作用
3. **黏膜保護**: 可能有助於黏膜修復

## 安全性考量

### 已知安全性資訊

1. **甲狀腺功能影響**: 長期高劑量可能導致甲狀腺功能異常
2. **碘過敏**: 對碘過敏者禁用
3. **妊娠哺乳**: 可穿透胎盤,需謹慎使用
4. **腎功能**: 腎功能不全者需調整劑量

### 藥物交互作用 (重要)

碘化鉀有多項重要藥物交互作用,需特別注意:

**Major (主要) 交互作用**:
| 交互藥物 | 機制/風險 |
|----------|-----------|
| ACE抑制劑 (Benazepril, Captopril, Enalapril等) | 增加高血鉀風險 |
| ARB類 (Losartan, Valsartan, Irbesartan等) | 增加高血鉀風險 |
| Eplerenone | 嚴重高血鉀風險 |
| Trimethoprim | 高血鉀風險 |

**Moderate (中等) 交互作用**:
| 交互藥物 | 機制/風險 |

### 建議

**鼻腔真菌感染**:
1. **適用族群**: 真菌性鼻竇炎,特別是接合菌病、腐黴病
2. **合併治療**: 建議與其他抗真菌藥物(如itraconazole)合併使用
3. **進一步研究**: 需要前瞻性臨床試驗評估療效和安全性
4. **臨床定位**: 作為難治性真菌性鼻部感染的輔助治療選項

**急性喉咽炎**:
- 缺乏直接證據支持
- 不建議作為主要治療選項

### 用藥監測建議

1. **甲狀腺功能**: 長期使用需定期監測TSH
2. **血鉀濃度**: 特別是合併使用高血鉀風險藥物時
3. **腎功能**: 定期監測肌酐清除率

### 限制說明

- 鼻腔真菌感染的證據主要來自病例報告
- 缺乏大規模隨機對照試驗
- 獸醫研究結果不能直接外推至人類
