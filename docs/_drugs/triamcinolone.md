# Triamcinolone：從過敏性疾病到黏蛋白性脫髮

## 一句話總結

Triamcinolone 是一種合成糖皮質激素，廣泛用於過敏性疾患、皮膚炎及關節炎等炎症性疾病，在台灣已有 20 張許可證上市。TxGNN 模型預測它可能對**黏蛋白性脫髮 (Alopecia Mucinosa)** 有效，目前有 **0 個臨床試驗**及 **4 篇文獻**支持這個研究方向，現有證據以疾病描述性個案報告為主，仍屬初步。

## 為什麼這個預測合理？

目前缺乏詳細的作用機轉資料。根據已知資訊，Triamcinolone 屬於糖皮質激素類別藥物，其消炎與抗過敏效果已在皮膚疾患、關節炎及過敏反應等多種適應症中獲得臨床驗證。糖皮質激素透過與細胞內受體結合，進一步抑制 NF-κB 等促炎轉錄因子，廣泛壓制淋巴球、嗜中性球及巨噬細胞的炎症浸潤。

黏蛋白性脫髮（Alopecia Mucinosa）是一種罕見的炎症性毛囊疾病，以毛囊及皮脂腺中黏多糖（黏蛋白）異常沉積、並伴隨炎性細胞浸潤為特徵，可為原發性（良性自限型）或繼發性（與皮膚 T 細胞淋巴瘤相關）。Triamcinolone 的消炎適應症與皮膚炎症病變高度重疊，理論上其強效的免疫抑制作用可改善毛囊周圍的局部炎症反應，進而緩解黏蛋白沉積相關病變。

然而，現有文獻均為疾病描述性個案報告，尚無直接以 Triamcinolone 治療 Alopecia Mucinosa 的前瞻性或對照研究資料。在建立更明確的機轉連結之前，本方向仍屬研究假說階段。

## 結論與下一步

**決策：Hold**

**理由：**
現有 4 篇文獻均為描述性個案報告（最新者發表於 2014 年），無任何臨床試驗登記，且文獻中並未直接評估 Triamcinolone 用於 Alopecia Mucinosa 的療效；TxGNN 預測的機轉假說雖有一定理論基礎，但作用機轉資料（MOA）亦付之闕如，整體證據強度（L4）不足以支持立即進入臨床開發。

**若要推進需要：**
- 補齊 Triamcinolone 的 DrugBank 作用機轉資料（MOA），確認與毛囊炎症路徑的具體關聯
- 設計前臨床研究（如毛囊黏蛋白症動物模型），驗證皮內注射或局部外用 Triamcinolone 的療效
- 以回溯性病例系列或前瞻性病例系列評估局部糖皮質激素在原發性 Alopecia Mucinosa 中的實際使用經驗
- 釐清原發性（良性自限型）與 CTCL 相關型之間的適應症區隔，避免在惡性病因未排除的情形下延誤適當治療
- 評估既有外用及注射劑型（台灣已上市）的給藥途徑是否適合皮膚局部治療應用
